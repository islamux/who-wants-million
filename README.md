# من سيربح المليون

لعبة "من سيربح المليون" مبنية باستخدام React + TypeScript + Vite. اللعبة تحتوي على 15 سؤالًا، كل سؤال له أربع إجابات، مع مساعدات مثل 50:50 وسؤال الجمهور.

## التقنيات

- React 19 + TypeScript 6
- Vite 8 (Rolldown bundler)
- Tone.js للتأثيرات الصوتية
- lucide-react للأيقونات
- PWA (تثبيت + تشغيل بدون إنترنت)

## البدء

```bash
git clone https://github.com/islamux/who-wants-million-vue2.git
cd who-wants-million-vue2
pnpm install
pnpm dev
```

## الأوامر

| الأمر | الوصف |
|-------|-------|
| `pnpm dev` | تشغيل خادم التطوير |
| `pnpm build` | بناء الإصدار النهائي مع PWA |
| `pnpm preview` | معاينة الإصدار النهائي |
